# Projeto 633 — Jorge Silva & Nicole Carvalho
 
Scripts PowerShell para administração de um servidor Windows, feitos para a disciplina de Sistemas.
 
Para iniciar, corre o `Menu.ps1` como Administrador — ele dá acesso a tudo o resto.
 
## O que cada script faz
 
- **Menu.ps1** — menu principal, ponto de entrada
- **GestaoProcessos.ps1** — lista os 10 processos a usar mais RAM e permite encerrar um
- **MonitorizacaoDisco.ps1** — mostra o espaço em disco e avisa se estiver quase cheio
- **GestaoUtilizadores.ps1** — criar, remover ou listar utilizadores locais
- **MonitorizacaoRede.ps1** — ping, verificação de portas e configuração de IP
- **ConfigurarRede.ps1** — configurar a rede para DHCP ou IP estático
- **SegurancaLogs.ps1** — verifica tentativas de login falhadas nas últimas 24h
- **BackupSistema.ps1** — faz backup da pasta `C:\Dados` para `C:\Backups`
